# Zansphere Assessment – Chatbot Prototype

## Overview
This project is a simple chatbot prototype built as part of the Zansphere Full Stack Developer Intern assessment.

The goal of this task was to identify a real problem and build a practical solution with clear thinking and implementation.

---

## Problem Statement
While exploring the Zansphere website, I observed that users do not have an instant way to get answers to their queries.

Users may need to:
- Search manually through the website
- Wait for email responses
- Navigate multiple pages

This creates friction in user experience and may lead to user drop-off.

---

## Proposed Solution
To solve this problem, I built a **simple chatbot interface** that provides instant responses to common user queries.

The chatbot helps users:
- Get quick answers
- Improve interaction
- Save time

---

## Features
- Interactive chat interface
- Instant responses to user input
- Simple and clean UI
- Works directly in browser (no setup needed)

---

## Tech Stack
- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic)

---

## How It Works
The chatbot uses a rule-based approach:

- It reads user input
- Matches keywords (like "services", "contact")
- Returns predefined responses

Example:
- Input: "services"
- Output: "Zansphere offers web and AI solutions"

---

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double click on `index.html`
4. The chatbot will open in your browser
